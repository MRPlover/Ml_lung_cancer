Данная работа посвящена заблаговременному оповещению о возможности наличия раковой опухоли в лёгких.
Файлы:
- app.py (Файл с программой)
- App_ML.ipynb (Блокнот с запуском программы)
- Project_slider:
	- cancer_patient_data_sets.csv (Файл с базой данных для одного типа ввода данных для предсказаний)
	- ML_prog.ipynb (Обработка данных, обучение моделей и т.д.)
	- Остальные файлы с сохранёнными обученными моделями, кодировщиками и масштабированием.
- Project_Yes_No:
	- survey lung cancer.csv (Файл с базой данных для второго типа ввода данных для предсказаний)
	- ML_prog_2.ipynb (Обработка данных, обучение моделей и т.д.)
	- Остальные файлы с сохранёнными обученными моделями, кодировщиками и масштабированием.
